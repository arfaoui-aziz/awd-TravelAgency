# AWD - Travel Agency
AWD - Travel Agency is a mini project developed to practice the  Microservice Architecture

## Topics covered
- Microservice Architecture
- Docker
- Spring Cloud
- Discovery Server : Eureka Server
- Spring Security
- Api Gateway + Filters : Zuul

## Microservices List 
- [x] Hotel
- [] Flight
- [] Excursion
